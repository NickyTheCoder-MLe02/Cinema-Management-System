# Cinema-Management-System
Project 08 - Database Management System

## Objective
A database management system developed to efficiently manage movies, cinema rooms, screening schedules, customers, and ticket reservations. 

## Tech Stack
* **Database Management System:** MySQL
* **Backend Application:** Python (mysql-connector-python)
* **Frontend Prototype:** ReactJS (Deployed on Vercel)

## Project Structure
* `/cinema_project`: Contains the Python CLI application (`main.py`, `db_connection.py`, `operations.py`).
* `/cinema-frontend`: Contains the ReactJS source code for the frontend prototype.
* `Dump20260501.sql`: The full MySQL database schema and sample data.

## Features
* **Database Design:** Normalized relational schema with foreign key constraints.
* **Advanced Database Objects:** Utilizing Indexes for speed, Views for daily reporting, Stored Procedures for secure ticket booking, and Triggers to prevent overbooking.
* **Security:** Role-based access control (Admin vs. Ticket Clerk) to protect customer Personally Identifiable Information (PII).
* **Application:** A fully functional Python Command-Line Interface to interact with the database.
